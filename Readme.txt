I. Folder Contents:-

1. Project Code(.ipynb files,datasets,models)
2. Project Documents(project papers,synopsis,presentation,final report)
3. Readme(guide)

II. Software Requirements:-

1.Tensorflow version-3.11(for Jupyter Notebook only)
2.Jupyter Notebook
OR
3.Any web browser to run Google Colab

III. Steps to run the project:-

1.Open the terminal and run Jupyter Notebook
OR
2.Open the Google Colab.
3.Upload and open the .ipynb files (1_Toddler_ModelTraining & 2_Child_ModelTraining).
4.Upload the datasets from input folder.
5.Upload the model files for each algorithm from respective folders for child and toddler.
6.Clear the output and restart the notebooks.
7.Run all cells to get the desired output.
